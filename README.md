# Alfred-Notion

This Ruby code is called by [Alfred](https://www.alfredapp.com/) workflow to interact with Notion API to create a note in the Notion database.

# Requirement

* [Alfred](https://www.alfredapp.com/)
* [Notion](https://www.notion.so/product)
